#### 1000. 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.



\[입력]

```
1 2
```

\[출력]

```
3
```

\[정답]

<pre class="language-python"><code class="lang-python"><strong>A, B = map(int, input().split())
</strong>print(A + B)
</code></pre>

\[해설]

* map(function, iter) : 순서대로함수와 그 객체를 받아 적용한다.
